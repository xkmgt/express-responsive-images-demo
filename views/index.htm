DEMO express-responsive-images

bird.jpg has a dimension of 4000x2000px and is stored in /public/images folder.

In the /public directory you will find a newly created folder: images-cache. Resize the window, reload the page and see the /public directory what happened.

Code: <img src="/images/bird.jpg">

This image is scaled to the window width and is cached in /public/images-cache/...

Code: <img src="/images/bird.jpg?w=200">

This image is scaled to 200px in width and is cached in /public/images-cache/200/bird.jpg

Code: <img src="/images/bird.jpg?w=400">

This image is scaled to 400px in width and is cached in /public/images-cache/400/bird.jpg
